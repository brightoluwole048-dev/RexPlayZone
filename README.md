# RexPlayZone - Website Package

This package contains a ready static website for RexPlayZone.

Files:
- index.html (homepage)
- style.css
- script.js
- games.json (data)
- games/ (game pages)
- assets/ (placeholder images)

To deploy:
1. Upload all files to your GitHub repo root (or unzip into your host).
2. If using GitHub Pages, ensure branch `main` and folder `/(root)` selected in Settings â†’ Pages.
3. Wait ~30 seconds and visit: https://<your-username>.github.io/RexPlayZone/

Edit:
- Update games.json to add or change games.
- Replace images in assets/ with real thumbnails.
